Mithrandir Firewall

Firewall is a fundamental part of a network, its function is to monitor incoming and outgoing network traffic and decide to allow or block such traffic according to a set of predefined security rules. However, this function relies on constant management to update or create new rules to avoid breaches and security breaches. With this, the proposed work aims to provide an easy-to-use web application to optimize the configuration and management of a firewall.
